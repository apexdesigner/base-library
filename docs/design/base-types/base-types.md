# Base Types

The base-types directory defines application-level type wrappers with validation and formatting constraints.

The [email](email.base-type.md), [phone](phone.base-type.md), [url](url.base-type.md), and [zip code](zip-code.base-type.md) base types provide validated string formats for common contact and address fields. The [uuid](uuid.base-type.md) base type wraps universally unique identifiers.

The [currency](currency.base-type.md) and [percentage](percentage.base-type.md) base types handle numeric values with appropriate formatting. The [duration](duration.base-type.md) base type represents time spans. The [calendar date](calendar-date.base-type.md) base type wraps date values without a time component.

The [json](json.base-type.md) base type stores arbitrary JSON data. The [semantic version](semantic-version.base-type.md) base type validates version strings following the semver convention.
